# Enchant Abbreviation 26

Standalone 26.1.2 Fabric client mod for SkyBlock enchanted book abbreviations, extracted from the Skyblocker idea.

## What it does

- Detects SkyBlock enchanted books with a single enchantment in `custom_data.enchantments`.
- Shows the Skyblocker-style abbreviation in the top-right of the item slot.
- Normal enchants render blue; ultimate enchants render light purple.

## Build

Upload this folder to GitHub and run the included Actions workflow. Use the normal jar from `build/libs`, not the `sources.jar`.

Target:
- Minecraft 26.1.2
- Fabric Loader 0.19.3
- Fabric API 0.152.1+26.1.2
- Java 25
